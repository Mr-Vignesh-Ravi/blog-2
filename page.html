Difference between Doucument and Window Obejects

The Document Object:

The document object represents the web page loaded in the browser. It’s like a backstage pass to the content within the HTML document. Here are some key points about the document object:

Accessing the Document:

    You can access the document object using either window.document or simply document.
    It provides a model representation of the currently loaded document, including elements like headings, paragraphs, images, and links.
    You can access the document object using either window.document or simply document.

Commonly Used Properties of document

    activeElement: Returns the currently active element in the document.
    body: Gives you access to the contents of the element.
    anchors: Provides all elements with a name attribute.
    baseURI: Represents the base URI of the document.
    cookie: Retrieves the cookie associated with the current document.
    And many more!

The Window Object:

The window object represents the browser window itself. It’s like the control center for your web page. Here’s what you need to know about the window object:

  • Properties Related to the Window:
  • Properties like length, innerWidth, innerHeight, and caches are stored in the window object
  • It’s the first thing that gets loaded into the browser when you open a page.

    • Accessing the Document via the Window:

  • The window object has a property called document, which stores a representation of the document.
  • So, when you access window.document, you’re essentially diving into the content of the loaded page.
